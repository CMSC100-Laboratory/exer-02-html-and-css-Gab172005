Steps on how to create github page
1. Create a github repository (It must be public)
2. Add a readme file
3. Add an index.html file and commit changes
4. Go to your repository Settings
5. Click pages
6. Under Build and deployment, ensure the "Source" is set to "Deploy from a branch" and the branch is set to main
7. Go live and wait about 1-2 minutes for your link

I learned how to create a website using only html and css and deployed it using github pages
